# 📚 UAS_RPL_PERPUSTAKAAN2026

Aplikasi **Perpustakaan Digital** berbasis **CodeIgniter 4** yang dibuat untuk memenuhi tugas **UAS Rekayasa Perangkat Lunak (RPL)** tahun 2026.

Aplikasi ini dirancang untuk membantu pengelolaan data perpustakaan seperti data buku, anggota, peminjaman, pengembalian, serta sistem login multi-user.

---

## 🚀 Fitur Utama

✅ Login User & Admin  
✅ Register Akun Baru  
✅ Dashboard Admin  
✅ CRUD Data Buku  
✅ CRUD Data Anggota  
✅ Peminjaman Buku  
✅ Pengembalian Buku  
✅ Laporan Data Buku  
✅ Tampilan Modern & Responsive  

---

## 🛠️ Teknologi yang Digunakan

- PHP 8+
- CodeIgniter 4
- MySQL / MariaDB
- Bootstrap 5
- HTML, CSS, JavaScript
